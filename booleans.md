# Boolean algebra:

### not
#### not a
| a     | result |
|-------|--------|
| False | True   |
| True  | False  |


### or
#### a or b
| a     | b     | result |
|-------|-------|--------|
| False | False | False  |
| True  | False | True   |
| False | True  | True   |
| True  | True  | True   |


### and
#### a and b
| a     | b     | result |
|-------|-------|--------|
| False | False | False  |
| True  | False | False  |
| False | True  | False  |
| True  | True  | True   |

- - -

# Boolean expressions 
* <     less than
* >     greater than
* ==    equal to
* <=    less than or equal to
* >=    greater than or equal to
* <>    not equal to
* not
* or
* and

```
print 1 < 2
>>> True
print 1 > 2
>>> False

print 1 < 5 < 10
>>> True
```
