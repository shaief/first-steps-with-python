# A list of examples of float usages:

## add numbers:
```
print 1.0 + 1.25
>>> 2.25
```
## subtract numbers:
```
print 9.5 - 8.0
>>> 1.5
```
## multiply numbers:
```
print 2.0 * 3.5
>>> 7.0
```
## divide numbers:
```
print 8.0 / 4.0
>>> 2.0
```

## what happens when dividing a number that have
## for example:
```
print 5.0 / 3.0
>>> 1.6666666666666667
```
## if we want only the part:
```
print 5.0 // 3.0
>>> 1.0
```
## to get the only the remainder:
```
print 5.0 % 3.0
>>> 2.0
```
## 2 power 3:
```
print 2.0 ** 3.0
>>> 8.0
```
